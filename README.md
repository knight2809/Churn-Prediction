Customer Churn PredictionThis repository contains a Logisitic Regression implementation from scratch using Python and NumPy to predict customer churn for a service-based company. The model identifies whether a client will stop using the service based on several behavioral and demographic features.📊 Dataset OverviewThe dataset includes client information with the following attributes:FeatureDescriptionNamesName of the customerAgeCustomer AgeTotal_PurchaseTotal Ads PurchasedAccount_ManagerBinary (0=No manager, 1=Account manager assigned)YearsTotal years as a customerNum_SitesNumber of websites using the serviceOnboard_dateDate the latest contact was onboardedLocationClient HQ AddressCompanyName of Client CompanyChurnTarget Variable (1=Churned, 0=Stayed)🛠️ Project Workflow1. Data PreprocessingFeature Selection: Dropped non-numeric and high-cardinality features such as Names, Location, Company, and Onboard_date.Feature Scaling: Utilized StandardScaler to normalize the feature space, ensuring the gradient descent converges efficiently.Data Splitting: 80/20 train-test split to validate model performance.2. Implementation from ScratchRather than using high-level libraries like Scikit-Learn for the model, this project implements the core logic of Logistic Regression:Sigmoid Function: Maps predictions to probabilities between 0 and 1.Cost Function: Log-Loss (Binary Cross-Entropy) to measure model error.Gradient Descent: Optimization algorithm used to update weights ($\theta$) over 1000 iterations.3. Training & PerformanceThe model was trained with a learning rate ($\alpha$) of 0.01.Accuracy Score: ~88.89% on the test set.Convergence: The cost history plot shows a steady decrease in error, stabilizing as it approaches the final iterations.🚀 UsagePrerequisitesPython 3.xPandas, NumPy, Matplotlib, Seaborn, Scikit-LearnRunning the ProjectClone the repository.Ensure customer_churn.csv and customers_stop.csv are in the project directory.Run the Jupyter Notebook Churn.ipynb.Prediction on New DataThe model includes a pipeline to process new, unseen data (customers_stop.csv).
